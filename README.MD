# Project MiniLinks

## Using:

- `/shorten`

  - #### **Creates unique short id for the entered url**

- `/{short_id}`
  - #### **Find original url by the short_id**

## Instruments:

- Sqlite3
- FastAPI
- Poetry

## How to run server:

```bash
uvicorn app.main:app --reload
```
